# Overseas - Study Abroad Consulting Website

## Overview
Overseas is a comprehensive website designed for a study abroad consulting business. The platform helps students connect with educational consultants who can guide them through the process of applying to universities abroad, obtaining visas, and preparing for international education opportunities.

## Features
- **Home Page**: Introduction to services with testimonials and featured destinations
- **Services**: Detailed information about university applications, visa guidance, and educational counseling
- **Destinations**: Information about popular study destinations including the USA, UK, Canada, and more
- **About Us**: Company history, mission, vision, and team information
- **Contact Form**: Interactive form for prospective clients to reach out for consultations
- **Testimonials**: Success stories from previously assisted students
- **Blog/Resources**: Educational content about studying abroad

## Technology Stack
- **Frontend**: Next.js with React
- **Styling**: Tailwind CSS
- **Deployment**: Vercel/GitHub Pages
- **Forms**: Form handling with validation
- **Responsive Design**: Mobile-first approach ensuring compatibility across devices

## Getting Started

### Prerequisites
- Node.js (v14.0.0 or later)
- npm or yarn

### Installation
1. Clone the repository
   ```
   git clone https://github.com/Tarunya0704/overseas.git
   ```

2. Navigate to the project directory
   ```
   cd overseas
   ```

3. Install dependencies
   ```
   npm install
   # or
   yarn install
   ```

4. Start the development server
   ```
   npm run dev
   # or
   yarn dev
   ```

5. Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.

## Customization

### Styling
The project uses Tailwind CSS for styling. You can modify the color scheme and other design elements by editing:
- `tailwind.config.js` - For theme customization
- `styles/globals.css` - For global styles

### Content
Update the website content by modifying:
- Text content in components and pages
- Images in the `public` directory
- Data files in the `data` directory

## Deployment
The project is configured for easy deployment on platforms like Vercel or GitHub Pages.

### Deploy on Vercel
1. Fork the repository to your GitHub account
2. Connect your GitHub repository to Vercel
3. Follow the prompts to deploy your website
